# 变更记录

## 0.8.1

- 优化容器、镜像、数据卷和网络列表的悬停反馈，左侧定位条会在鼠标跨行移动时连续跟随，并适配系统减少动态效果偏好。

## 0.8.0

- 修复 Docker Context 切换失败后仍覆盖活动连接的问题；候选连接现在仅在探测成功后提交，失败时继续使用原 Context。
- 概览最近操作改为按日期分组，并修复状态圆点放大动画被列表边界裁切的问题。
- Context 新增和编辑改用带进退场动画的公共模态弹窗，输入框尺寸与网络连接弹窗保持一致。
- 所有原生下拉框替换为适配应用主题的自定义组件，支持键盘导航、焦点状态和长列表浮层定位。
- macOS 窗口启用保留系统交通灯的沉浸式标题栏，并补充顶部拖拽区域。

## 0.7.0

- 镜像列表新增运行容器入口，可读取镜像默认 Entrypoint、Cmd、Env、工作目录、用户、暴露端口和卷信息，并通过表单创建并启动容器。
- 新增 `InspectImageRunConfig` 和 `RunImage` 后端 API，容器运行流程使用 `CORIVA_IMAGE_RUN` 日志关键字串联排障信息。
- 镜像声明端口会在运行表单中预填展示，默认不发布到宿主机，需用户主动勾选后才会配置 Host IP 或 Host Port。

## 0.6.2

- 网络清理面板标题新增应用内问号提示，说明清理会删除未被容器使用的 Docker 网络，并可通过 Filters 缩小范围。

## 0.6.1

- 修复从网络容器列表发起断开连接时会提前关闭容器列表弹窗的问题。

## 0.6.0

- 支持点击网络列表中的容器数量打开已连接容器弹窗，并可直接对单个容器执行断开网络操作。

## 0.5.0

- 网络列表新增容器使用数量展示，便于快速识别每个 Docker network 当前关联的容器规模。

## 0.4.1

- 将新建 Docker network 表单改为模态弹窗，避免挤压网络列表内容。
- 修复网络表单和设置表单中复选框不可见、不可正常勾选的问题。

## 0.4.0

- 新增 Docker network 管理能力，支持列表、搜索、创建、详情、连接容器、断开容器、删除和清理未使用网络。
- 新增网络高级参数支持，包括 labels、driver options、IPAM 配置、connect driver options、网关优先级和 prune filters。
- 新增 `CORIVA_NETWORK_ACTION` 日志关键字，用于串联网络管理操作的完整执行链路。
